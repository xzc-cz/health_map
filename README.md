# 🗺️ HealthMap 健康地图应用

**HealthMap** 是一个基于 Mapbox 的 Android 应用程序，旨在帮助用户记录和查看每日健康活动计划，并通过地图进行可视化展示。  
它鼓励用户养成健康习惯，也让社区用户之间的运动和计划信息变得可见，促进社交互动与健康意识提升。

---

## 📱 核心功能

- 📝 **健康计划记录**：用户可以录入多个每日健康活动，包括时间、活动内容和位置。
- 📅 **日期筛选**：通过日期选择器，仅显示特定日期的计划。
- 🗺️ **地图展示**：通过 Mapbox 地图以标签形式显示用户的所有计划。
- 🧠 **情境感知（开发中）**：整合 Google 日历，实现基于用户事件的个性化体验。
- 🌍 **多语言支持（预留）**：设计支持未来国际化扩展。

---

## 📷 页面概览

| 页面              | 描述说明                                         |
|-------------------|--------------------------------------------------|
| 🏠 主页面         | 显示入口按钮，跳转至计划录入或地图页面           |
| 📝 计划录入页面   | 输入健康计划的表单（支持多条计划）               |
| 🗺️ 地图展示页面 | 显示所有用户健康计划的地图视图，可点击标签        |
| 📅 日期选择器     | 筛选显示特定日期的计划                            |

## 🛠️ 技术栈

- Kotlin + Jetpack Compose
- Mapbox SDK v11.11.0
- Android SDK：minSdk=26，targetSdk=35
- Material 3 UI 组件
